# Projeto mecatrônica

## Objetivos

O objetivo deste projeto é a digitação de notas de aula do curso de Mecatrônica da USP.

O texto está sendo elaborado por Vitor Martins usando [LaTeX][0] e as figuras por [Régis Santos][1] usando [TikZ][2].
